# 🐧 Simulado LPI Linux Essentials 010-160

Simulado interativo para preparação para a certificação **Linux Professional Institute — Linux Essentials (010-160)**, desenvolvido em HTML, CSS e JavaScript puro — sem dependências externas além do Chart.js.

---

## 📋 Sobre o Projeto

Este simulado foi criado para ajudar estudantes a se prepararem para o exame **LPI Linux Essentials 010-160**, cobrindo todos os 5 tópicos oficiais do currículo da certificação com questões baseadas no material didático oficial da LPI.

---

## ✨ Funcionalidades

- 🎓 **Modo Prova** — Simula as condições reais do exame: cronômetro sem pausa, sem feedback durante a prova e resultado detalhado ao final
- 🛠️ **Modo Prática** — Ideal para aprender: explicação de cada alternativa, possibilidade de pausar e marcar questões para revisar
- 📊 **Gráfico de Desempenho** — Radar chart com histórico de erros por tópico
- 🚩 **Marcação de Questões** — Marque questões para revisar depois
- 🔍 **Filtro por Tópico** — Estude um tópico específico de cada vez
- 🌙 **Tema Claro/Escuro** — Alterne entre os temas a qualquer momento
- ✕ **Cancelar sem afetar estatísticas** — No modo prática, cancele sem comprometer o gráfico de desempenho
- 💾 **Histórico persistente** — O desempenho é salvo localmente no navegador
